Meteor UIkit
==================
[UIKit](http://www.getuikit.com) 3.x packaged for Meteor forked from [meteor-uikit](https://github.com/alexandesigner/meteor-uikit)

### To install
``meteor add alexandesigner:meteor-uikit``
